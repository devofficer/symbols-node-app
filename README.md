# Symbols Assessment

Simple HTML Generator

## Description

A Simple node app generates HTML from json object file.

## Getting Started

### Dependencies

```
"fs": "^0.0.1-security"
```

### Installing

```
npm install
```

### Executing program

```
npm start myobject.txt
or
node index.js myobject.txt
```
